# Solar Corrective Work Order – Team Members Pane (from View)

Ανοίγει το pane **Maintenance Team Members** από τη λίστα (view) του πίνακα Solar Corrective Work Order.

## Αρχεία
- `corrective-workorder-view-team-members-pane.js` – Script από command bar του view.
- `corrective-workorder-team-members-pane.html` – Κοινό HTML pane.

## Προαπαιτήσεις
- Web resources ανεβασμένα και published στο Solardev.
